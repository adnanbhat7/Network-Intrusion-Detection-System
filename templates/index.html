Architecture

How It Works

Every network connection generates a flow record. ShieldNet analyses it through a five-stage pipeline.

01
Packet Capture

Raw packets are intercepted at the network interface. CICFlowMeter converts bidirectional flows into structured records containing timing, size, and flag statistics.

02
Feature Extraction

50 statistical features are computed: forward/backward packet rates, inter-arrival times, payload size distributions, TCP flag counts, and 4 engineered ratio features validated by Mutual Information scoring.

03
RobustScaler Normalisation

Features are scaled using IQR-based normalisation. RobustScaler is chosen over StandardScaler because attack traffic produces extreme outliers that distort mean-based scaling.

04
CNN Inference

A 1D-CNN (Conv1D 32→64 filters, GlobalAveragePooling, Dense 128→64, Dropout) processes the normalised feature vector and outputs a 4-class softmax probability distribution.

05
Firewall Verdict

The dominant class determines action. Benign traffic passes. DoS, BruteForce, and Botnet traffic is blocked, the source is quarantined, and an alert is logged.

Technical Details

Model & Dataset

Trained on CIC-IDS 2017/2018. Optimised for Macro F1 to handle severe class imbalance.

Dataset

CIC-IDS 2017/2018. Four classes: Benign, DoS, BruteForce, Botnet. Severe imbalance — Benign ≫ Botnet — addressed with BorderlineSMOTE and SMOTETomek oversampling.

Architecture

1D-CNN: Conv1D(32,k=6) → Conv1D(64,k=3) → MaxPool → GlobalAvgPool → Dense(128, drop=0.4) → Dense(64, drop=0.3) → Dense(4, softmax). L2 regularisation on dense layers.

Feature Selection

3-stage funnel: VarianceThreshold removes zero-signal columns, Mutual Information ranks by class-discriminative power, correlation pruning drops features with r > 0.95.

Evaluation

Primary metric: Macro F1. Accuracy is misleading with class imbalance — a naive majority classifier scores high accuracy but near-zero Macro F1. ROC-AUC (OvR, macro) used as a confidence check.

Baselines

RandomForest, XGBoost, LightGBM, ExtraTrees, and GradientBoosting across four resampling strategies. CNN evaluated separately with the same preprocessing pipeline.
